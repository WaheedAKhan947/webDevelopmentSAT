Case Sensitivity

Statement: Declare, Assign, Compute
Declare: Use let, const, or var to create variables.
Assign: Assign values to variables (x = 10;).
Compute: Perform operations (let sum = x + 20;).

Comments
Single-line: //.
Multi-line: /* */.

Dynamically Typed Language
Variable types are determined at runtime.

Example:

let value = "Hello"; // String
value = 42; // Number

Types of DataTypes
Primitive:
String, Number, Boolean, Undefined, Null.
User-Defined:
Objects, Functions
Boolean, Undefined , String, Null, number 

Variables: Memory
Variables are storage locations in memory.
Use let, const, or var to declare variables.

Naming Conventions
camelCase: myVariableName.
PascalCase: MyVariableName.
snake_case: my_variable_name.
Valid characters: $, _, a-z, A-Z, 0-9.

Keywords
Reserved words (e.g., if, let, function) cannot be used as variable names.
Example: let class = "math"; // Error.
